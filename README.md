# Color Guess
Um jogo que consiste em acertar qual é a cor com base nos valores RGB.

# Tecnologias usadas:
- HTML
- Javascript
- CSS

# Instalação
- Clone o repositório: "git clone https://github.com/ThiagoMarchini/color-guess.git"
- Navegue até o diretório e abra o arquivo "index.html"
